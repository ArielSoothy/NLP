1.1 Dataset Analysis & Emotion Distribution

📊 What we analyzed:

Analyzed the emotion dataset and created a comprehensive distribution analysis of emotions in our training data.

Data Analysis Code 
import pandas as pd
import matplotlib.pyplot as plt

# Load and analyze emotion distribution
df = pd.read_csv('emotion_dataset.csv')
emotion_counts = df['emotion'].value_counts()

# Create pie chart
plt.figure(figsize=(10, 8))
plt.pie(emotion_counts.values, labels=emotion_counts.index, autopct='%1.1f%%')
plt.title('Emotion Distribution in Dataset')
plt.show()

🎯 Key Finding

Severe class imbalance detected: 82.1% neutral emotions vs only 1.9% hate. This imbalance requires special handling during training to ensure the model learns to detect minority emotions effectively.

1.2 Why Precision & Recall Beat Accuracy

🤔 The Problem with Accuracy

With such imbalanced data, accuracy becomes misleading. A model could achieve 82% accuracy by always predicting "neutral" but would be completely useless for emotion detection.

Metric Definition Problem with Accuracy Why Precision & Recall Matter
Accuracy Correct predictions / Total predictions 82% by always predicting "neutral" Misleading for imbalanced data
Precision True Positives / (TP + False Positives) 0% - never detects emotions Shows actual detection quality
Recall True Positives / (TP + False Negatives) 0% - misses all emotions Shows coverage of actual emotions

💡 Key Insight

Precision and recall force the model to actually learn emotion patterns rather than just exploiting class distribution. They measure the model's ability to correctly identify each emotion class.
